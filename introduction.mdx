Terarium is a web-based platform built for the [DARPA ASKEM program](https://www.darpa.mil/research/programs/automating-scientific-knowledge-extraction-modeling) that empowers researchers to work with complex computational models and simulations. It combines a visual workflow canvas, AI-assisted modeling tools, and scientific simulation engines into a single collaborative environment.

## What you can do with Terarium

<Steps>
  <Step title="Organize your research in projects">
    Create projects to group related models, datasets, documents, and workflows. Share projects with collaborators and manage access through roles.
  </Step>
  <Step title="Import and build models">
    Upload models from files (AMR, SBML, STELLA, MDL formats), extract them from scientific papers using AI, or create them directly from mathematical equations. Edit model structure, parameters, and metadata in the visual editor.
  </Step>
  <Step title="Compose workflows on the canvas">
    Use the drag-and-drop workflow canvas to chain operations — load a model, configure its parameters, run a simulation, calibrate against real data, and visualize results — all as a connected, reproducible pipeline.
  </Step>
  <Step title="Simulate and analyze">
    Run forward simulations, calibrate model parameters against observed data, optimize intervention policies, and validate model behavior with FUNMAN — all powered by the CIEMSS simulation engine.
  </Step>
</Steps>

## Key capabilities

| Capability | Description |
|---|---|
| **Visual workflows** | Compose multi-step modeling pipelines on an interactive canvas |
| **AI assistance** | Extract models from documents, enrich metadata, and configure parameters with AI |
| **CIEMSS simulation** | Probabilistic simulation, calibration, optimization, and ensemble analysis |
| **MIRA integration** | Stratify, transform, and compare models using the MIRA modeling framework |
| **Knowledge extraction** | Parse equations, tables, and text from scientific PDFs |
| **Beaker notebooks** | Run Python code alongside models in an integrated notebook environment |
| **FUNMAN validation** | Verify that model parameters satisfy scientific constraints |

<Note>
  Terarium is built for researchers working on complex systems modeling. It assumes familiarity with differential equation models, scientific simulation, and concepts from epidemiology, biology, or physics depending on your domain.
</Note>
